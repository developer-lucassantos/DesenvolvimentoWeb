# Athos Desenvolvimento Web

Site institucional da **Athos Desenvolvimento Web**, empresa focada em soluções digitais para negócios de delivery, incluindo **PDV**, aplicativos e sistemas de gestão operacional.

## Estrutura
- `index.html` — página principal
- `style.css` — estilos do site
- `script.js` — interações do menu mobile

## Publicação
Este projeto pode ser publicado facilmente com **GitHub Pages** usando a branch `main` e a pasta `/root`.

## Objetivo
Apresentar a empresa e suas soluções:
- PDV para delivery
- Aplicativo próprio para pedidos
- Painel administrativo
- Estrutura digital para operações comerciais
